# Security Policy

## Supported Versions

| Version     | Supported          |
| ----------- | ------------------ |
| Pre-release | :white_check_mark: |

## Project Status

**Notice:** The Neotic project is currently in a **pre-release** phase. We are actively testing out features, architectures, and integrations. As a result, certain components may still be unstable, and security vulnerabilities may be present. We deeply appreciate your patience and assistance in making the project secure.

## Reporting a Vulnerability

If you discover a security vulnerability or have concerns regarding the security of this project, please do **not** open a public issue. 

Instead, please contact our core maintainers directly for assistance and responsible disclosure:

- [@memer0](https://github.com/memer0)
- [@abhintr2006](https://github.com/abhintr2006)
- [@toxicbishop](https://github.com/toxicbishop)

We take all security reports seriously and will collaborate with you to resolve the issue as quickly as possible during this testing phase. Thank you for keeping Neotic safe!
